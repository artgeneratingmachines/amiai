Description

Are you bored of static cartoon pictures as NFTs? How about an art machine that interacts with you in real-time? You can freely try from amiai.art. This is not art itself; it is a dynamic art-generating machine fully contained in a single HTML file for sale.

This art piece is turn-based gameplay between human and AI. Thus the series is named "Am I AI?". It is a philosophical question we ask ourselves all over human history. Yet, this first piece attacks AI using AI tools. The MNIST dataset has been used in place of pixel intensity values; however, it does not use any AI model to generate the pictures. It also behaves like a mirror converting the camera stream into numbers which is the reflection of reality between the digital world and the real world. It also reminds us of the precise calculations that are happening in our bodies. Thus, the first piece is named "We ourselves are art".

The lookup table space is also on sale. You can own your 28x28 matrix and place your NFT to shape the final drop. As a spoiler, the next move is coming from AI "Are we dreaming ?". Are you ready to be part of the matrix?

How to Use

User interface is minimal in this application. Start button starts the video camera. Load button opens file dialog to load an image. Then, current resolution would be used to generate image. You can increase or decrease resolution with Res buttons. You can save the results as a PNG file with save button.
